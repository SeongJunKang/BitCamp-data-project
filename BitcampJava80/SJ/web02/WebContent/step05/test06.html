text:
일반적인 텍스트 입력상자
name을 지정하지 않으면 서버에 보내지 않는다.
값을 입력하지 않으면 문자열을 보낸다.
텍스트 상자의 값을 지정하려면 value 속성을 사용하라.

password:
암호를 입력할 때 주로 사용한다. 입력 텍스트가 보이지 않는다.

checkbox: 스릴러 SF 드라마
사용자에게 여러 개의 값을 선택하게 만들 때 사용한다
체크한 항목의 값이 파라미터로 서버에 전달된다.
체크하지 않으면 서버에 전달되지 않는다.
value 속성에 값을 지정하지 않으면 기본 값이 "on"이다.

radio: 검정 빨강 파랑 장미 백합 진달래
같은 이름을 가진 라디오 버튼끼리 배타적 선택을 할 수 있다.

file:
form 태그의 enctype을 "multipart/form-data"로 지정해야만 서버에 파일을 보낼 수 있다.

button:
일반 버튼은 자바스크립트로 리스너를 등록하지 않는 한 특별한 동작을 하지 않는다.

reset:
폼 안에 있는 모든 항목을 초기 상태로 설정한다.

submit:
서버에 get/post 요청을 보낸다.
action 속성에 URL을 지정하지 않으면 현재 URL을 다시 요청한다.

color:
서버에 값을 보낼때 #rrggbb 형식으로 보낸다.
단 "#" 문자는 URL 인코딩해서 보내기 때문에 %23이 될 것이다.
서버 개발자는 신경쓰지말라! 서버에서는 %23이 다시 "#"으로 바뀔 것이다.

date:
날짜 값은 yyyy-MM-dd 형식으로 보낸다.

selected 속성 이름만 있어도 해당 항목이 선택된다.
option의 value를 지정하지 않으면 option 태그의 콘텐츠가 사용된다.
multiple 속성을 붙이면, 한 번에 여러 개의 값을 선택할 수 있다.
Ctrl/Shift 키를 이용하여 여러개의 값을 선택할 수 있다.
서버에 보낼 때는 같은 이름으로 선택된 값을 여러 개 보낸다.
예) state2=2&state2=4
